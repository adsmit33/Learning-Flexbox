# Flexbox Directions
## Important takeaways
#### Flex-direction
* row (Main Axis) - left to right
* row-revers (Main Axis) - right to left
* column (Cross Axis) - top to bottom
* column (Cross Axis) - bottom to top 

```css
.container {
  flex-direction: row | row-reverse | column | column-reverse;
}
```
